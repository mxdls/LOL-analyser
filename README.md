## 项目简介 ##
项目目的是实现对lol战绩的爬取，然后对战绩进行分析，训练一个用于预测战局胜利方的机器学习模型。<br>
战局信息用MySQL存储。神经网络初步使用keras+tensorflow后端进行训练。

## 使用方法 ##
**user：**<br>
- 按用户名查找用户<br>
- 更新用户信息<br>

**battle：**<br>
- 按用户和区获取战绩列表<br>
- 获取战局详细信息<br>

**spider：**<br>
- 随机按用户爬取对局信息，保存到数据库<br>
- 多线程爬取数据，包括战局信息和用户信息<br>

**query：**<br>
- 显示统计信息<br>

**make_train_test：**<br>
- 生成供网络训练用的二进制文件，参数n为测试集大小<br>
